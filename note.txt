# origin
원격 저장소를 지칭하는 이름으로, git clone, git pull, git push 등을 수행 할 때, 별도의 이름을 지정하지 않으면 붙는 이름

# git pull
원격 저장소의 변경된 내용을 지역 저장소로 가져오고 싶을 때 사용 한다. 원격 저장소에서 브랜치를 생성한 경우, 생성한 브랜치도 가져 온다
